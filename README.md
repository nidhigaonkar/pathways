# Pathways - Infinite canvas chat with Perplexity API

An interactive infinite canvas app for exploring AI conversations through branching pathways. Create multiple chat nodes, merge conversation threads, interact with AI beyond a single chat interface.

## Features

- **Infinite Canvas**: Pan, zoom, and navigate through your conversation nodes
- **Branching Conversations**: Fork nodes to explore different conversation paths
- **Node Merging**: Combine multiple conversation threads with AI summaries
- **Parallel Execution**: Run multiple nodes simultaneously with batch queries
- **Speech-to-Text**: Voice input support for hands-free interaction
- **Search**: Find nodes quickly across your canvas
- **Real-time AI Responses**: Powered by Perplexity
